Relative or Absolute Links

The important difference between an 'absolute' and 'relative' hyperlink.

Absolute Hyperlink

http://www.website.com/index.html

Relative Hyperlink

index.html

What are they?

A link is an absolute link if the URL and file name can be found from anywhere on the Web, not just from a single Web site.

A relative link specifies the name of the file to be linked to only as it is related to the current document.

Appropriate Usage

An absolute link should only be used when linking outside of the current website.

A relative link should be used when linking within the website.
